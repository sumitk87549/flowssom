# Flowssom setup
1. Install Node.js 20 LTS.
2. `npm install`
3. Create `.env` with `EXPO_PUBLIC_SUPABASE_URL` and `EXPO_PUBLIC_SUPABASE_ANON_KEY`.
4. `npx expo start` (or `npm run web` for web).

## Manual configuration
- Create Supabase project and `rooms` table.
- Configure RevenueCat API key placeholders.
